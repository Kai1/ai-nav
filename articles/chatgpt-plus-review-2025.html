🔍 核心功能逐项评测

🧠
o3-mini 推理模型
Plus专属

OpenAI 最新发布的 o3-mini 是截至2025年4月最强数学推理模型之一。AIME 2024 得分 87%+（high模式），超过 DeepSeek R1（79.8%）和 Gemini Thinking（73.3%）。

三个推理深度档：low（速度快，适合简单问题）、medium（平衡）、high（最深度推理，适合竞赛数学/复杂逻辑）。

适合场景：STEM作业、竞赛数学、代码调试（逻辑推断）、复杂商业分析。不适合：写作、日常对话（用GPT-4o更快）。

💡 使用技巧
对话框左侧选"o3-mini"，然后点"思考强度：高"。复杂数学题响应约需30-60秒，但准确率显著高于GPT-4o直接回答。
🎨
DALL-E 3 图像生成
Plus专属

Plus用户每天约 50张 DALL-E 3 图像（每次生成1-4张）。DALL-E 3 最大优势是文字渲染准确（海报/标签类图片），以及与 ChatGPT 对话无缝集成——可以在对话中直接要求"再把人换成金发"连续修改。

与竞品对比：真实感比不上 FLUX.1，艺术风格不如 Midjourney v6.1，但使用门槛最低（不需要额外注册），文字排版是同类最准的。每月50张对轻度图片需求完全足够。

📊
高级数据分析（Code Interpreter）
Plus专属

这是 ChatGPT Plus 最被低估的功能。上传 Excel/CSV 文件，用自然语言提问即可："找出销售额最高的5个产品"、"画出过去12个月的增长趋势"、"帮我找数据中的异常值"。

ChatGPT 在沙盒中运行真实 Python 代码（Pandas + Matplotlib），生成的图表可直接下载。对比雇佣数据分析师（$50/h+），这个功能的ROI极高。

实测案例：上传一份1.2万行销售数据CSV（3.8MB），要求"按地区分析季度增长率并绘制热力图"，ChatGPT 在约45秒内完成完整分析，包括4张可视化图表。

🤖
Custom GPTs（GPT商店）
Plus专属

GPT Store 已有超过 300万个 Custom GPTs（2025年3月数据）。可以找到针对特定场景深度优化的专属AI：

  • Scholar AI：访问2亿+学术论文数据库，带引用
  • Consensus：基于科学文献回答问题，标注置信度
  • Code Copilot：代码生成专家，比默认GPT-4o更专注
  • SEO GPT：SEO内容优化，关键词分析
  • PDF Summarizer：PDF快速摘要，比上传文件更便捷

Plus用户还可以创建自己的Custom GPT：上传公司知识库、设定回复风格、连接外部API，无需编程。实际上是在打造私有AI助理。

🖊️
Canvas 协作写作
Plus专属

2024年10月发布的 Canvas 是一个内嵌的文档编辑界面，可以：在右侧预览文档全貌，选中特定段落要求AI修改，追踪AI建议的变更（类似Google Docs的建议模式），设定阅读水平（大学/高中/小学）。适合长篇文档迭代写作，比来回对话更直观。

🔄 2025年新增功能盘点

记忆功能升级（2024年底-2025年初）

ChatGPT 现在可以跨会话记住用户偏好：你的名字、职业、常用语言、写作风格偏好、之前讨论过的项目背景。Plus用户记忆容量更大，可存储更多条目。实测：告诉ChatGPT"我是Python开发者，倾向于简洁代码"，之后每次对话都会自动考虑这个背景。

Projects 功能（2024年12月）

类似 Claude 的 Projects，可以为不同项目创建独立上下文：上传项目相关文件、设定系统提示、保持独立对话历史。适合需要同时推进多个项目的用户。

ChatGPT Operator（2025年早期发布，Plus可用）

AI Agent功能，可以在浏览器中自主执行任务：填写表单、在网站搜索信息、预订机票。目前仍处于早期阶段，稳定性有限，但代表AI工具从"对话"向"自主行动"的重要转变。
