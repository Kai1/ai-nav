    ⚙️ AI SEO实战工作流：从关键词到排名

    第一步：关键词研究（Semrush / Ahrefs）

    目标：找到搜索量适中(1k~30k/月)、竞争度低(KD < 40)的关键词 流程： 1. Semrush Keyword Magic Tool 输入种子关键词 2. 筛选条件：KD < 40 + 搜索意图 = 信息类 3. 用AI语义聚类将关键词按主题分组 4. 导出关键词集群 → 制定内容日历 Ahrefs补充检查： - Site Explorer 分析竞品 TOP 关键词 - Content Gap 找出你有但竞品没有排名的词

    第二步：内容创作（Surfer SEO + ChatGPT）

    流程： 1. Surfer SEO Content Editor 输入目标关键词 2. 获取：推荐字数 / 关键词清单 / 竞品TOP10结构 3. 用 ChatGPT 生成初稿（参考大纲） 4. 在 Content Editor 实时优化，目标 Content Score ≥ 75 5. 人工审阅：添加独家数据、案例、专家观点 提示词模板： "写一篇[字数]字的文章，关键词是[目标词]， 包含以下章节：[章节列表]，语气[专业/轻松]， 面向[受众]，目标读者是[人群描述]"

    第三步：技术优化（Screaming Frog / Alli AI）

    核心检查清单： ✅ Core Web Vitals：LCP < 2.5s, CLS < 0.1, INP < 200ms ✅ 移动端友好：Mobile-Friendly Test通过 ✅ Schema标记：Article / Review / FAQ / HowTo ✅ 内部链接：每篇文章3~5个相关内部链接 ✅ 标题H1唯一，H2/H3语义层次清晰 ✅ 图片：WebP格式 + Alt文字 + 懒加载 ✅ 页面速度：GTmetrix 90分+ (Performance)
